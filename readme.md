# 🚀 Employee Performance Tracker

A simple and interactive **web-based application** to record, monitor, and analyze employee performance using **Python, Streamlit, and Pandas**.

---

## 📌 Overview

The **Employee Performance Tracker** helps organizations manage employee performance data efficiently. Users can add employee details, track monthly performance, and visualize insights through charts.

This project is ideal for beginners learning **data handling, visualization, and web app development** using Streamlit.

---

## ✨ Features

* ➕ Add employee performance records
* 📋 View all employee data in a structured table
* 📊 Department-wise performance analysis
* 📈 Monthly performance trends
* 🏆 Identify top-performing employees
* 💾 Data stored locally using CSV

---

## 🛠️ Tech Stack

* **Python** – Core programming language
* **Streamlit** – Web application framework
* **Pandas** – Data manipulation and analysis

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/employee-performance-tracker.git
cd employee-performance-tracker
```

---

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

---

### 3️⃣ Run the Application

```
streamlit run app.py
```

---

## 📊 Usage

1. Open the app in your browser
2. Use the sidebar menu:

   * **Add Employee** → Enter employee details
   * **View Data** → See all records
   * **Analytics** → View charts and insights
3. Data is automatically saved in `data.csv`
